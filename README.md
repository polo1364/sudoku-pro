# 🧩 Sudoku Pro+ 數獨遊戲（6 提示版）

穩定版 Sudoku，支援手機與電腦。  
重玩後提示次數自動重置為 **6 次**。

## 🎮 功能特色
- 🎚️ 難度：簡單、中等、困難
- 💡 提示次數 6 次（重玩會重置）
- 💾 自動儲存進度
- ⏱️ 計時系統
- ✅ 檢查錯誤標示
- 🔁 重玩 / 新題目
- 📱 響應式設計

## 🧠 遊玩方式
1. 每行、每列、每個 3×3 小宮中，數字 1–9 不可重複。
2. 點格輸入數字。
3. 功能按鈕：
   - 新題目：生成新局。
   - 提示：顯示正確數字（最多 6 次）。
   - 檢查：檢查正確性。
   - 重玩：清空盤面並重置提示次數。
4. 完成所有格且正確即完成。

## 📦 使用方式
開啟 `index.html` 即可離線遊玩。  
支援 GitHub Pages、Firebase、Netlify 部署。
